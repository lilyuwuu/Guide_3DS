    Homebrew Launcher (Steelhax)

    영어로 지원을 요청하고 싶으시면 Nintendo Homebrew Discord 서버에 도움을 요청해 주세요.
    이 가이드가 도움이 되었다면, 기부해 주시면 감사하겠습니다.

    목차

    • 준비물
    • 진행 방법
      • 섹션 I - 준비 작업
      • 섹션 II - Steelminer
      • 섹션 III - 하드웨어 버튼 작동 확인
    • 다음 단계: 익스플로잇 선택
    기술적 상세 정보 (선택 사항)

    더 깊은 모딩을 위한 Seedminer의 이 사용법은, Steel Diver: Sub Wars의 커스텀 세이브 파일을 movable.sed 파일을 사용해서 작성해, Homebrew Launcher를 실행하는 방식입니다.

    이 과정은 Steel Diver: Sub Wars의 기존 저장 데이터를 덮어쓸 것입니다. 만약 게임의 저장 데이터를 보존하고 싶으시다면 00000001.sav 파일을 백업해 두시는 것을 권장합니다.

    준비물

    +콘솔에 설치된 게임 “Steel Diver: Sub Wars”

    • 이 게임을 다운로드 한 적이 있다면, 다시 다운로드 할 수 있습니다
    • Steel Diver: Sub Wars를 설치하려면 SD 카드가 들어가 있어야 합니다
    • Seedminer에서 받은 movable.sed 파일
    • 최신 버전의 Steelhax (직접 다운로드)
    • 최신 버전의 Luma3DS (.zip 파일)
    • otherapps.zip (직접 다운로드)

    진행 방법

    섹션 I - 준비 작업

    이 섹션에서는, Homebrew Launcher를 실행할 파일들을 복사합니다.

    1. 콘솔을 종료해 주세요
    2. SD 카드를 컴퓨터에 삽입해 주세요
    3. Luma3DS .zip안에 있는 boot.firm과 boot.3dsx를 SD 카드의 루트에 복사해 주세요
      • SD 카드의 루트는 SD 카드를 열 때 보이는 디렉토리이고, Nintendo 3DS 폴더가 보이지만 그 안에 들어가 있지 않은 상태의 폴더입니다
    4. Steelhax-release.zip 안에 있는 steelhax 폴더를 SD 카드의 루트에 복사해 주세요
    5. 콘솔의 모델/지역/버전에 해당한 otherapp 페이로드를 otherapps.zip에서 SD 카드의 루트에 복사하고, 이 파일을 otherapp.bin으로 이름을 바꿔 주세요
      • .bin 확장자는 보이지 않는 이상 추가하지 말아 주세요
    6. ‘movable.sed’ 파일을 SD 카드의 루트에 복사해 주세요
    7. SD 카드를 콘솔에 다시 삽입해 주세요
    8. 콘솔을 시작해 주세요

    섹션 II - Steelminer

    이 섹션에서는, Steel Diver: Sub Wars의 해킹된 저장 파일을 생성하고, 이걸 사용해서 Homebrew Launcher를 실행합니다.

    1. Launch “Steel Diver: Sub Wars” once and select a Mii to initialize your save file
      • Decline the game update when prompted
      • If you have already updated the game, use the Data Management menu of the System Settings to delete the update
    2. Exit “Steel Diver: Sub Wars”
    3. 콘솔을 종료해 주세요
    4. SD 카드를 컴퓨터에 삽입해 주세요
    5. SteelHax 저장 툴 웹사이트를 컴퓨터에서 열어 주세요
    6. movable.sed 파일을 선택해 주세요
    7. Select your region based on your console
    8. “Build and Download”를 선택해 주세요
    9. 완료될 때까지 기다려 주세요
    10. SD 카드에서 Nintendo 3DS -> <ID0> -> <ID1>-> title -> 00040000 -> <8-글자-지역> -> data 폴더로 들어가 주세요
      • EUR 지역: 000d7e00
      • JPN 지역: 000d7c00
      • USA 지역: 000d7d00
    11. 새로 다운로드한 00000001.sav 파일을 data 폴더로 복사해 주세요
      • 만약 표시가 된다면, 전 세이브 파일을 덮어쓰세요
    12. SD 카드를 콘솔에 다시 삽입해 주세요
    13. 콘솔을 시작해 주세요
    14. Launch “Steel Diver: Sub Wars”
    15. 익스플로잇이 성공적이었다면 Homebrew Launcher로 부팅되었을 것입니다
    16. 콘솔을 종료해 주세요

    섹션 III - 하드웨어 버튼 작동 확인

    이 섹션에서는 콘솔의 안전 모드 기능을 사용 할 수 있는지 테스트할 것 입니다. 이 테스트를 통해, 다음 페이지에서 어떤 방법을 취해야 할지 결정할 수 있습니다.

    이 섹션에서는 콘솔의 좌/우 어깨 버튼 (L, R 버튼)이 정상적으로 동작하는지 확인합니다. 이 테스트를 통해, 다음 페이지에서 어떤 방법을 취해야 할지 결정할 수 있습니다.

    1. 콘솔을 시작해 주세요
    2. HOME 메뉴가 보이면, (L 버튼)과 (R 버튼)을 동시에 눌러주세요
      • 카메라 앱이 실행되어야 합니다
    3. 콘솔을 종료해 주세요

    다음 단계: 익스플로잇 선택

    이전 섹션에서 카메라가 표시되었다면, 안전 모드가 정상적으로 실행될 수 있을 것입니다.

    카메라가 실행되었다면, boot9strap 설치 (HBL-USM)으로 계속합니다

    카메라가 실행되지 않았다면, boot9strap 설치 (Frogtool)로 계속합니다
