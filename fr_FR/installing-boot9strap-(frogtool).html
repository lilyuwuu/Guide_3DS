    Détails techniques (facultatif)

    For technical details on the exploits that you will be using on this page, see “DSiWare_Exports CTCert verification” and “FIRM partitions known-plaintext” here.

    To use the magnet link for frogcert.bin on this page, you will need a torrent client like qBittorrent or Deluge.
    If you are unable to install one, follow Bannerbomb3 instead.

    Ce dont vous avez besoin

    • Your movable.sed file from completing Seedminer or nimhax on the root of your SD card
    • frogcert.bin (magnet link)
    • The latest release of Frogtool
    • The latest release of b9sTool

    Section I - Vérification de la présence d’un CFW

    Comme mesure de sécurité supplémentaire, nous effectuerons une vérification supplémentaire de la présence d’un custom firmware. Ceci est dû au fait que l’utilisation de cette méthode lorsqu’un custom firmware est déjà installé à un risque de bricker la console (ce qui la rendrait inutilisable sans méthodes de récupération comme ntrboot).

    1. Éteignez votre 3DS
    2. Maintenez le bouton (Select) enfoncé
    3. Allumez votre console tout en maintenant le bouton (Select) enfoncé
    4. Si la vérification a réussi, vous démarrerez sur le menu HOME et vous pourrez continuer avec ce guide
    5. Éteignez votre 3DS

    Si vous voyez un menu de configuration ou si la console s’éteint immédiatement, vous avez déjà CFW, et continuer avec ces instructions peut BRICKER votre console ! Suivez Vérification de la présence d’un CFW pour mettre à jour votre CFW existant.

    Section II - Préparatifs

    1. Éteignez votre 3DS
    2. Insérez votre carte SD dans votre ordinateur
    3. Copiez boot.firm et boot.3dsx depuis Luma3DS .zipvers la racine de votre carte SD
    4. Copy boot.nds (b9sTool) from the b9sTool release .zip to the root of your SD card
    5. Copy Frogtool.3dsx to the /3ds/ folder on your SD card
    6. Copy frogcert.bin to the root of your SD card
    7. Retirez la carte SD de votre ordinateur et réinsérez-là dans votre 3DS
    8. Allumez votre console

    Section III - Patching DS Download Play

    1. Open the Homebrew Launcher through your method of choice (likely the Internet Browser)
    2. Launch Frogtool from the list of homebrew
    3. Select the “INJECT patched DS Download Play” option
    4. Frogtool will automatically run and inject the JPN version of Flipnote Studio into your DS Download Play
    5. Once this operation has finished, read the screens and check if the process was successful
    6. Tap the touch screen, then select “BOOT patched DS Download Play”
    7. Si l’exploit a réussi, votre console devrait avoir démarré la version japonaise de Flipnote Studio
      • If you get stuck on a black screen, follow this page, then try again

    Section IV - Flipnote Exploit

    Si vous préférez un guide visuel pour cette section, un tel guide est disponible ici.

    Dans cette section, vous allez effectuer une série d’étapes très spécifiques dans Flipnote Studio qui, une fois effectuées correctement, déclencheront le lancement l’installateur de boot9strap (custom firmware).

    1. Terminez le processus de configuration initial de Flipnote Studio jusqu’à ce que vous atteigniez le menu principal
      • Sélectionnez l’option de gauche chaque fois que vous y êtes invité durant le processus de configuration
    2. En utilisant l’écran tactile, sélectionnez la grande case à gauche, puis sélectionnez la boîte avec une icône de carte SD
    3. Une fois le menu lancé, sélectionnez l’icône visage, puis le bouton en bas à droite pour continuer
    4. Appuyez sur (X) ou (Haut) sur la croix directionnelle en fonction de ce qui est affiché sur l’écran supérieur
    5. Sélectionnez le deuxième bouton en haut avec une icône de bobine de film
    6. Défilez vers la droite jusqu’à ce que la bobine “3/3” soit sélectionnée
    7. Touchez la troisième case contenant la lettre “A”
    8. Défilez vers la gauche jusqu’à ce que la bobine “1/3” soit sélectionnée
    9. Touchez la quatrième case contenant la lettre “A”
    10. Si l’exploit a réussi, votre console devrait avoir démarré b9sTool
    11. Séléctionnez “Install boot9strap” en utilisant la croix directionnelle
      • Si vous manquez cette étape, la console retournera au menu HOME au lieu d’installer boot9strap et vous devrez ouvrir Connexions Nintendo DS et recommencer à partir du début de cette section
    12. Appuyez sur (A), puis appuyez sur START et SELECT en même temps pour démarrer l’installation
    13. Une fois que l’installation est terminée et que vous voyez le message “done.” s’afficher sur l’écran du bas, quittez b9sTool, puis éteignez votre console
      • Vous devrez peut-être forcer l’arrêt de la console en maintenant le bouton Power enfoncé
      • Si votre console s’éteint lorsque vous essayez de l’allumer, assurez-vous d’avoir copié le fichier boot.firm depuis l’archive .zip de Luma3DS à la racine de votre carte SD
      • Si vous voyez l’écran de configuration de Luma, éteignez votre console et passez à la section suivante

    Section VI - Luma3DS Configuration

    1. Your device should have booted into the Luma3DS configuration menu
      • Le menu de configuration de Luma3DS sert à régler les paramètres du custom firmware Luma3DS. Beaucoup de ces paramètres peuvent être utiles pour la personnalisation ou le débogage
      • For the purpose of this guide, leave these options on the default settings (do not check or uncheck anything)
      • Si votre console s’éteint lorsque vous essayez de l’allumer, assurez-vous d’avoir copié le fichier boot.firm depuis l’archive .zip de Luma3DS à la racine de votre carte SD
    2. Appuyez sur (Start) pour enregistrer les modifications et redémarrer

    À partir d’ici, votre console démarrera sur Luma3DS par défaut.

    • Luma3DS ne présente aucune différence visuelle avec le menu HOME normal. Si votre console a démarré dans le menu HOME, elle exécute un custom firmware.
    • Sur la page suivante, vous installerez des applications homebrew utiles pour compléter votre configuration.

    Section VII - Restoring DS Connection Settings

    1. Lancez l’application Mode téléchargement
    2. Attendez jusqu’à ce que vous voyiez les deux boutons
      • N’appuyez sur aucun des deux boutons
    3. Appuyez simultanément sur (L) + (Bas sur la croix directionnelle) + (Select) pour ouvrir le menu Rosalina
    4. Sélectionnez “Miscellaneous options”
    5. Sélectionnez “Switch the hb. title to the current app.”
    6. Appuyez sur (B) pour continuer
    7. Appuyez sur (B) pour revenir au menu principal de Rosalina
    8. Appuyez sur (B) pour quitter le menu Rosalina
    9. Appuyez sur (Home), puis fermez Mode téléchargement
    10. Relancez l’application Mode téléchargement
    11. Votre console devrait maintenant charger le Homebrew Launcher
    12. Launch Frogtool from the list of homebrew
    13. Select the “RESTORE patched DS Download Play” option
    14. Once this operation has finished, read the screens and check if the process was successful
      • If there are any errors or missing files, correct the problem and try again
    15. If the process was successful, tap the touch screen, then press START to exit
    16. Éteignez votre 3DS
